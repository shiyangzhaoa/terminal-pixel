## Install

```shell
npm install terminal-pixel -g
```

## How to use

```shell
terminal-pixel
// or npx terminal-pixel
```

```
> Pixel Center
  Game Center
/Users/xx/bad-apple.mp4

-------------- or --------------

  Pixel Center
> Game Center

> snake

Use the 'esc' key to exit the game
```
